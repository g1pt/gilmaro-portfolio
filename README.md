# Gilmaro Piter Portfolio

Recruitergerichte portfolio website voor junior data-, automation- en systemsrollen.

## Positionering

- Self-Taught Data & Automation Engineer
- Research Systems Builder
- Junior Data & Systems Specialist

De website is gericht op House of Bèta, data traineeships, junior data analyst rollen, applicatiebeheer, research engineering, automation rollen en software service providers.

## Tech stack

- React
- Vite
- CSS
- Netlify-ready static build

## Lokaal draaien

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Netlify

- Build command: `npm run build`
- Publish directory: `dist`

## CV

Plaats je CV handmatig in:

```text
public/cv/Gilmaro_Piter_CV.pdf
```

De site gebruikt deze downloadlink:

```text
/cv/Gilmaro_Piter_CV.pdf
```

Omdat er lokaal geen CV-PDF is gevonden, toont de site nu: `CV beschikbaar op aanvraag`.
